# 郭津硕 Guo Jinshuo - Personal Portfolio

这是我的个人作品集网站，展示我的个人信息、教育背景、研究兴趣等内容。

## 在线访问

网站已部署在 GitHub Pages，可以通过以下地址访问：

- **GitHub Pages**: https://jinshuoguo.github.io
- **自定义域名**: https://jinshuoguo.github.io

## 项目结构

```
├── index.html          # 主页面
├── editor.py           # 页面编辑器脚本
├── avatar.jpg          # 个人头像
└── README.md           # 项目说明
```

## 本地开发

1. 克隆仓库
```bash
git clone https://github.com/jinshuoguo/jinshuoguo.github.io.git
cd jinshuoguo.github.io
```

2. 使用任意本地服务器预览
```bash
# 使用 Python
python -m http.server 8000

# 或使用 VSCode Live Server 等插件
```

3. 访问 `http://localhost:8000`

## 部署

本项目使用 GitHub Pages 自动部署。推送到 `main` 分支后，网站会自动更新。

```bash
git add .
git commit -m "描述你的更改"
git push
```
